Για να εκτελέσετε τα αρχεία μέσω του τερματικού (Terminal, Command Prompt), με την εντολή "cd monopati/tou/fakelou" επιλέξτε το μονοπάτι του φακέλου στον οποίο βρίσκονται τα αρχεία. Έπειτα, για να εκτελέσετε τα αρχεία, εκτελέστε την εντολή "python onoma_arxeiou.py" για Windows συστήματα και "python3 onoma_arxeiou.py" για συστήματα τύπου Linux.

Προκείμενου να εκτελεστεί σωστά το αρχείο "erotima2.py", φροντίστε να συμπεριλάβετε το αρχείο "points.txt" στον ίδιο φάκελο.
